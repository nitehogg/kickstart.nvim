## Cheatsheet

#### VIM

| Command | Description                                                            |
| ------- | ---------------------------------------------------------------------- |
| yiw     | yank inner word                                                        |
| viwp    | visual inner work pase (replace inner word with previously yanked one) |
| }       | jump to next blank line                                                |
| {       | jump to prev blank line                                                |
| .       | repeat last motion                                                     |

#### Tmux

| Command    | Description   |
| ---------- | ------------- |
| ctrl-b, :  | command mode  |
| ctrl-b, up | previous pane |
